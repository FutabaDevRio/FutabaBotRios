## 📦 Instalación

```bash
git clone https://github.com/FutabaDevRio/FutaabaBotRios.git
cd FutaabaBotRios
npm install
npm start
Escaneá el código QR con WhatsApp y listo. El bot se ejecuta donde corra Node.js (Version De Node 20) (Windows, Linux, macOS, o cualquier VPS).
